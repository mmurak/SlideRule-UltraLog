Basic Operations

Move cursor to specified value on any scale at stator.
> Name-of-scale + [SPACE] + Value + [ENTER]
   E.G.: Move cursor to 5.75 on D. $\rightarrow$ D 575[ENTER]

Move slider to set the value on any scale at slider to the cursor.
> Name-of-scale + [SPACE] + Value + [ENTER]
   E.G.: Move slider and 2.7 on C to the cursor. $\rightarrow$ C 2.7[ENTER]

Move cursor to specified value on any scale at slider.
> [H] + Name-of-scale + [SPACE] + Value + [ENTER]
   E.G.: Move cursor to 2.3 on C. $\rightarrow$ HC 23[ENTER]

Operations around Slider Indices

Move slider to set the left index to the cursor.
> [LX] + [ENTER]

Move slider to set the right index to the cursor.
> [RX] + [ENTER]

Move cursor to the left index of the slider.
> [HLX] + [ENTER]

Move cursor to the right index of the slider.
> [HRX] + [ENTER]

Alignment Operations

Align slider with stator (Cursor stays on stator)
・$\uparrow$

Align stator with slider (Cursor stays on slider)
・$\downarrow$

Convenient Operations

Move cursor to the relevant index of the slider.
> [=]
   E.G.: Calculate $3\div 4$. $\rightarrow$ D 3 [ENTER] C 4 [ENTER] [=]

Fix cursor or slider to the relevant position in case of 'Out of Range.'
> [/]
   E.G.: Calculate $9\div 7\times 8$. $\rightarrow$ D 9 [ENTER] C 7 [ENTER] HC 8 [ENTER] [/]

Miscellaneous Operations

Flip over the slide rule.
> [TAB]

Clear input area.
> [ESC]

Scroll cursor to left/right.
> [$\leftarrow$] or [$\rightarrow$]
(You can press [SHIFT] to increase scroll amount.)
